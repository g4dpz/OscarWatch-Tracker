These shortcuts work on the main window while you are tracking or browsing. They do not apply when another dialogue (such as Settings) has focus.

Frequency panel

Requires a selected satellite with transponder data in the frequency overlay. See Frequencies.

KeysAction
Ctrl + W Toggle Voice / CW on linear SSB modes (same as the header buttons). Only when the Voice/CW row is shown.
Numpad + Increase the active offset (RX or TX, per the toggle) by 10 Hz (same as the +10 step button). Updates CAT when tracking is active.
Numpad − Decrease the active offset by 10 Hz (same as the −10 step button).

Numpad + / − and S are ignored while you are typing in a text box or the offset spinner, so you can still enter a value manually. Use the RX / TX toggle to choose which leg the keys adjust. Use Store offset to save an offset for next time — the keys only nudge the live value.

World map and sky plot

Click a satellite to select it, or use the keyboard after focusing the control. See Map & sidebar.

KeysAction
S Toggle solo view on the map and sky plot — hide all satellites except the selected one. Press again to show all.
Tab Move focus to the world map or sky plot (along with other controls).
← or ↑ Select the previous satellite in the list (when the map or sky plot has focus).
→ or ↓ Select the next satellite.
Enter or Space If nothing is selected yet, select the first visible satellite.

Upcoming Passes list

Use Tab to reach the pass list, then arrow keys to highlight a row and Enter to select that satellite — standard list behaviour.

Menu bar (Windows)

Press Alt and the underlined letter to open a menu, then choose an item with the keyboard. Examples: Alt+S for Satellites, Alt+H for Help (including Open logs folder and Open recordings folder). While in Standby, Alt+R opens Rotator manual control when that menu is visible.

Tip: On macOS, menu shortcuts use the Command key where the app provides them; global offset (RX or TX) and Voice/CW keys apply when the main window is active.
