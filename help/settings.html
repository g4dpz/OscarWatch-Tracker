Open from the menu: Settings. Press Save when finished. Settings are stored in %AppData%\OscarWatch\settings.json.

Station

Your operating position: name (for labels), latitude, longitude, Maidenhead grid, and height above sea level in metres. Accurate height helps slightly with geometry; position is essential for passes and az/el.

Tracking

  • Minimum elevation — passes below this angle are hidden from predictions.
  • Prediction hours — how far ahead to compute passes.
  • TLE updates — when to fetch fresh orbital data automatically.

Appearance

Light, dark, or follow Windows theme. The sky plot follows the theme; the world map image stays light for readability.

Voice

Optional spoken alert when a satellite rises above a set elevation (e.g. “Alpha Oscar Zero Seven is rising”). Choose a voice and test it. On Linux you may need espeak-ng installed.

Rotator & Radio

See Radio & rotator for COM ports, rig type, and rotator options.

Cloudlog

If you use Cloudlog, enter your server URL, API key, and radio name. While tracking, OscarWatch can post current satellite uplink/downlink frequencies to the Radio API. Use Test connection before relying on it in the field.
