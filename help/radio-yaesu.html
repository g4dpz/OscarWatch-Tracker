Yaesu support covers the FT-847 as a one-box satellite radio, and several models used as downlink or uplink legs in a dual-radio station.

FT-847 (single radio)

The FT-847 is a dedicated satellite rig. Pick FT-847 in Settings → Radio, set the COM port, and match the radio’s CAT baud rate in the menu.

  • OscarWatch sets satellite/split layout and updates RX/TX for Doppler.
  • Use Full Doppler in the frequency panel for normal cross-band work.
  • On linear SSB, you can spin the receive dial to hunt the passband — see spinning the dial.

Dual radio — FT-817, FT-818, FT-991, FTX-1

These models do not appear as a single-rig type. Enable Settings → Radio → Dual radio and assign each leg a type and COM port. A typical station: FT-818 on downlink, IC-706 on uplink (see ICOM dual).

Setup checklist

  1. Tick Dual radio.
  2. Choose rig type and COM port for Downlink and Uplink.
  3. Match baud rate on each radio to OscarWatch (table below).
  4. Close other CAT programs on those ports.
  5. In the frequency panel, use Full Doppler (not RX-only or TX-only) unless you know you need otherwise.

Baud and port notes

ModelMenu settingSuggested baudNotes
FT-817 / FT-818 Menu #14 4800 (38400 also works) Simple CAT; on FM the dial may be locked during tracking — use Pause CAT to tune by hand.
FT-991 / FT-991A Menu 031 CAT RATE 38400 8N2, hardware RTS; use the USB CAT COM port.
FTX-1 CAT-1 RATE 38400 8N2, hardware RTS; use Enhanced COM (CAT-1), not CAT-2.

FT-817 / FT-818: downlink stuck, uplink tracks

  • Use Full Doppler in the frequency panel.
  • On linear SSB, tune the downlink, then pause ~1 s so passband trim is captured.
  • On FM, CAT may hold the dial — use Pause CAT if you need manual tuning.
  • If the radio “locks up” after heavy CAT, power-cycle the radio and avoid swapping COM ports mid-pass.
  • Try increasing CAT delay on the Yaesu leg under Settings → Radio if commands are missed.
Heavy CAT: Small Yaesu rigs can hang if another program or a wrong baud rate hammers the port. One program per COM port, correct menu baud, and a short delay often fixes mysterious drop-outs.

Receive-only modes

For ISS SSTV or beacons (uplink 0), OscarWatch updates the downlink leg only. See receive-only overview.
