ICOM satellite rigs use Main for receive and Sub for transmit. OscarWatch talks to the radio over CI-V on a USB serial port.

Supported single-radio models

ModelNotes
IC-910 / IC-910HClassic VHF/UHF satellite rig
IC-9700Default CI-V address 94 (hex A2) unless changed in the radio
IC-9100HF/VHF/UHF; satellite mode as on other ICOM satellite rigs
IC-821HOlder satellite rig; Main/Sub only (no split CAT). Set uplink CTCSS on the radio manually

Settings → Radio

  • Rig type — pick your model.
  • COM port — the CI-V USB port (check Device Manager on Windows).
  • CI-V address — must match the radio (often 94 on IC-9700; IC-910 is often 7A). Wrong address = no CAT.
  • Baud rate — match the radio menu (often 19200 on IC-910; IC-9700 USB is often 115200).
  • CI-V TRN — should be On on the radio so commands are accepted.

What OscarWatch does on a pass

  • Turns satellite mode on for normal cross-band satellites.
  • Puts downlink on Main, uplink on Sub.
  • Updates both frequencies for Doppler and applies your offsets.
  • Keeps you on Main while you turn the dial to hunt SSB — uplink CAT waits briefly so Sub is not selected by accident. See spinning the dial.

Receive-only (ISS SSTV, beacons)

When uplink is 0, OscarWatch turns satellite mode off, puts the downlink on Main, and only tracks receive. See receive-only overview.

Dual radio — IC-705 or IC-706 as one leg

IC-705 and IC-706 / IC-706MKII / IC-706MKIIG are not full single-rig types in OscarWatch. They appear only when Dual radio is enabled — typically as the uplink or downlink leg paired with another radio (e.g. FT-818 on RX, IC-706 on TX).

Enable dual radio

  1. Settings → Radio → Dual radio — tick the box.
  2. Set Downlink and Uplink rig types and COM ports (each leg needs its own port).
  3. Save and connect — status should show both legs.

IC-705 (one leg)

  • CI-V USB Port = Link to [CI-V] in the radio menu.
  • Baud — often 115200; match OscarWatch.
  • CI-V address — default A4 unless you changed it.

IC-706 / IC-706MKII / IC-706MKIIG (one leg)

  • CI-V via the REMOTE jack (USB adapter).
  • Pick the correct leg type in the list — addresses differ: 48H (706), 4CH (MKII), 58H (MKIIG).
  • Baud 19200; CI-V TRN On on the radio.
Tip: After using another rig on the same USB lead, quit OscarWatch, unplug/replug USB, and restart if CI-V stops responding.

Common issues

  • Radio stuck on Sub — press Resume, or toggle satellite mode on the radio. See Problems?.
  • Uplink lags after tuning — normal for ~2.5 s after you stop moving Main; uplink then catches up.
  • Frequency snaps back when tuning near TCA — short settle period after CAT; try pausing CAT briefly or see Doppler tuning.
