The TS-2000 is supported as a single satellite radio. Unlike ICOM rigs, OscarWatch does not turn satellite mode on for you — you must do that on the radio before tracking.

Before each pass (or once per session)

  1. On the TS-2000, select SAT mode and the correct satellite VFO layout (VFO A / VFO B as you normally operate).
  2. In OscarWatch: Settings → Radio — rig type TS-2000, correct COM port and baud.
  3. Pick the satellite and transponder mode in the frequency panel.
  4. Press Resume (not Standby) and start tracking.

What OscarWatch controls

  • Receive and transmit frequencies (Doppler and your offsets).
  • Modes where the driver supports them for the selected transponder.
  • CTCSS on FM satellites when configured.

OscarWatch assumes you have already entered the satellite operating mode on the radio. If CAT updates the wrong VFO, check SAT mode and which VFO is receive vs transmit on the TS-2000 front panel.

Tuning tips

  • Linear SSB — use passband trim on the receive VFO; Doppler follows your offset.
  • Adjust Doppler thresholds if tracking feels sluggish near TCA.
  • Pause CAT in Settings if you want full manual control for a while.

Receive-only

For modes with uplink 0 (SSTV, beacons), OscarWatch updates receive frequency only. You may leave SAT off if you prefer simple VFO tuning — see receive-only overview.
